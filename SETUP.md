# 🚀 Инструкция по настройке бота с Google Drive + Sheets

## Что изменилось в боте

### Новая команда `/hw` — сдача ДЗ с выбором модуля
1. Ученик нажимает `/hw`
2. Видит кнопки с модулями (только несданные)
3. Выбирает модуль → отправляет файлы/фото/видео/текст
4. Нажимает «Готово» → бот записывает ДЗ

### Автоматизация
- Файлы → **Google Drive** (папка по имени ученика)
- Галочка → **Google Sheets** (✅ + дата/время)
- Пересылка куратору — как и раньше

---

## Шаг 1: Создать сервисный аккаунт Google (5 минут)

1. Зайди на https://console.cloud.google.com
2. Создай новый проект (например, «Yelumio Bot»)
3. В меню слева → **APIs & Services** → **Enable APIs**
4. Включи две API:
   - **Google Drive API**
   - **Google Sheets API**
5. В меню → **APIs & Services** → **Credentials**
6. Нажми **Create Credentials** → **Service Account**
7. Дай имя (например, `yelumio-bot`) → **Done**
8. Нажми на созданный сервисный аккаунт
9. Вкладка **Keys** → **Add Key** → **Create new key** → **JSON**
10. Скачается файл — переименуй его в `service_account.json`
11. Положи файл в папку с ботом (рядом с `bot.py`)

**Запиши email сервисного аккаунта** — он выглядит так:
`yelumio-bot@yelumio-bot-123456.iam.gserviceaccount.com`

---

## Шаг 2: Подготовить Google Drive (2 минуты)

1. Зайди на https://drive.google.com
2. Создай папку «Домашние задания»
3. Нажми ПКМ → **Share** (Поделиться)
4. Вставь email сервисного аккаунта → роль **Editor**
5. Скопируй ID папки из URL:
   ```
   https://drive.google.com/drive/folders/ЭТОТ_ID_КОПИРУЙ
   ```

---

## Шаг 3: Подготовить Google Sheets (3 минуты)

1. Создай новую таблицу на https://sheets.google.com
2. Переименуй лист в **Домашки** (важно — точное имя!)
3. Заполни шапку:

| A              | B    | C    | D    | E    | F    | G    | H    | I    |
|----------------|------|------|------|------|------|------|------|------|
| Имя ученика    | ДЗ 0 | ДЗ 1 | ДЗ 2 | ДЗ 3 | ДЗ 4 | ДЗ 5 | ДЗ 6 | ДЗ 7 |

4. **Можешь заранее вписать имена учеников в колонку A** (как они в Telegram).
   Если не впишешь — бот добавит их автоматически при первой сдаче ДЗ.
5. Расшарь таблицу на email сервисного аккаунта → роль **Editor**
6. Скопируй ID таблицы из URL:
   ```
   https://docs.google.com/spreadsheets/d/ЭТОТ_ID_КОПИРУЙ/edit
   ```

---

## Шаг 4: Вставить ID в бот

Открой `bot.py` и заполни 3 переменные:

```python
BOT_TOKEN              = "твой_токен_бота"
GOOGLE_SHEET_ID        = "ID_таблицы_из_шага_3"
GOOGLE_DRIVE_FOLDER_ID = "ID_папки_из_шага_2"
```

Или задай через переменные окружения:
```bash
export BOT_TOKEN="7992712058:AAF..."
export GOOGLE_SHEET_ID="1abc..."
export GOOGLE_DRIVE_FOLDER_ID="1xyz..."
```

---

## Шаг 5: Установить зависимости

```bash
pip install aiogram apscheduler google-api-python-client google-auth
```

---

## Шаг 6: Запустить

```bash
python bot.py
```

Если `service_account.json` не найден — бот всё равно запустится,
просто без Google-интеграции (будет работать как раньше).

## Как это выглядит для ученика

```
Ученик: /hw

Бот: 📤 Сдача домашнего задания
     Выбери, к какому модулю сдаёшь ДЗ 👇
     [🎯 ДЗ 0 — Введение]
     [💡 ДЗ 1 — Идея и концепция]
     [✍️ ДЗ 2 — Текст и промпты]
     ...

Ученик: *нажимает ДЗ 2*

Бот: 📝 Сдача ДЗ к Модулю 2 — Текст и промпты
     Отправь файл, фото, видео или текст!
     [✅ Готово, я всё отправил]
     [❌ Отмена]

Ученик: *отправляет 3 скриншота*

Бот: 📎 Файл принят (1 файл(ов))
Бот: 📎 Файл принят (2 файл(ов))
Бот: 📎 Файл принят (3 файл(ов))

Ученик: *нажимает "Готово"*

Бот: ✅ ДЗ принято!
     Модуль: Модуль 2 — Текст и промпты
     Файлов: 3
```

**В это время автоматически:**
- 3 файла загружены в Google Drive → `Домашние задания / Иван Иванов / M2_photo_*.jpg`
- В Google Sheets в строке «Иван Иванов», колонка «ДЗ 2» появилось: `✅ 16.03 14:32`
